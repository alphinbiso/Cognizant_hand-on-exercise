E-commerce Platform Search Function

Objective

Implement Linear Search and Binary Search algorithms to search products in an e-commerce platform and compare their performance.

Technologies Used

Java,VS Code,JDK 21

Time Complexity

| Algorithm | Best | Average | Worst |
|----------|------|---------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

Conclusion

Binary Search is more efficient for large sorted datasets, whereas Linear Search is suitable for small or unsorted datasets.
